# WATCHER FOLDER

- nodeJS

## INSTALL

1. Install packages
```bash

> npm install

```

2. create .env and insert the instructions
```bash

URL=   // URL TO FETCH (https://example.com/sendFile)
PORT=3000 // PORT FOR USING APPLICATION
WATCHDIR=  // FOLDER TO WATCHING FILES

```

## START YOUR APPLICATION

```bash
npm start
```
